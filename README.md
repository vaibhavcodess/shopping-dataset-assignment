# Shopping Dataset Data Cleaning Assignment

## Objective

The objective of this assignment was to perform data exploration and data cleaning using Python and Pandas.

## Tasks Performed

* Loaded the CSV dataset using Pandas
* Explored dataset using head(), tail(), shape, columns, and data types
* Identified and handled missing values
* Removed duplicate rows
* Cleaned the final_price column
* Created derived columns: quantity and total_amount
* Saved the cleaned dataset as a new CSV file

## Tools Used

* Python
* Pandas
* Google Colab

## Output

A cleaned dataset named `cleaned_dataset.csv` was generated successfully.
